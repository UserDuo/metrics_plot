# Citation Management

## Practices
- Use consistent citation style (e.g., Nature, APA, IEEE)
- Verify bibliographic metadata (authors, title, venue, year, DOI/URL)
- Prefer authoritative sources
- Maintain a reference list and in-text citation mapping

## Outputs
- Formatted references section
- In-text citation mapping
- Quality flags for each reference
