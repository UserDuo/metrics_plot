# Statistical Rigor

## Practices
- Assumption checks (normality, variance, independence)
- Use Welch tests when variances differ
- Report effect sizes (Cohen's d, r)
- 95% confidence intervals
- Correct for multiple comparisons when needed
- Transparent reporting (n, means, SD, test stats, p, CI)

## Output Fields
- Test used and justification
- Effect size and interpretation
- CI bounds and sample sizes
- Notes on violations and remedies
