# Literature Review Workflow

## Goals
- Define precise research questions and sub-questions
- Plan search queries and venues (journals, conferences, preprints)
- Screen relevance and quality
- Synthesize themes and methods
- Identify gaps and future directions

## Process
1. Scope & Questions
2. Query Planning (keywords, synonyms, boolean operators)
3. Venue & Source Prioritization (A* journals, top conferences, reputable preprints)
4. Screening (title/abstract, methods, datasets)
5. Thematic Synthesis (approaches, results, limitations)
6. Structured Summary (context, contributions, gaps)

## Output Structure
- Context and motivation
- Prior approaches (taxonomy)
- Comparative assessment
- Open challenges and gaps
- Implications for current study
