# Reproducibility Checklist

## Items
- Data availability and licensing
- Code availability (modules, scripts)
- Environment specification (versions, hardware)
- Random seeds and determinism
- Exact commands to reproduce
- Artifact provenance and checksums

## Output
- Reproducibility section ready for manuscript
- List of commands and configuration items
